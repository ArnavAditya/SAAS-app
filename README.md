# 🚀 SAAS App

A full-stack SaaS application built with a **client-server architecture**, featuring a modern frontend and a scalable backend.

---

## 📌 Project Structure

SAAS-app/
├── client/   # Frontend (React / Vite)
├── server/   # Backend (Node.js / Express)


---

## ⚙️ Tech Stack

### 🖥️ Frontend:

* React
* Vite
* JavaScript
* CSS

### 🔧 Backend:

* Node.js
* Express.js

---

## 🚀 Getting Started

1️⃣ Clone the repository


git clone https://github.com/ArnavAditya/SAAS-app.git
cd SAAS-app


---

### 2️⃣ Setup Backend

cd server
npm install
npm start

---

### 3️⃣ Setup Frontend

cd client
npm install
npm run dev


---

## 🌐 Environment Variables

Create a `.env` file in the `server` folder and add:

PORT=5000
